GitHub Pages Upload Guide

1. Repository name must be exactly:
   yourusername.github.io

2. Upload these files to the repository root:
   - index.html
   - style.css
   - .nojekyll
   - assets/ folder

3. In GitHub:
   Settings -> Pages
   Source -> Deploy from a branch
   Branch -> main
   Folder -> / (root)
   Save

4. Replace later if needed:
   - assets/avatar-placeholder.svg  -> your own headshot file
   - The GitHub link in the Contact section
   - Any section text you want to personalize further

5. Your CV is already included as:
   assets/cv.pdf
